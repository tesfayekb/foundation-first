# Action Tracker

> **Owner:** Project Lead | **Last Reviewed:** 2026-04-08

## Purpose

Tracks every completed action for audit trail and progress tracking.

## Scope

All tasks performed on the project.

## Format

| ID | Date | Action | Impact | Modules Affected | Docs Updated | Status |
|----|------|--------|--------|-----------------|-------------|--------|
| ACT-001 | 2026-04-08 | Created SSOT documentation system | HIGH | All | All governance, architecture, security, performance, module, quality, tracking, reference, planning docs | Complete |

## Rules

- Every completed task gets an entry
- Impact classification required
- Modules affected must be listed
- Docs updated must be listed
- No retroactive editing of completed entries

## Dependencies

- [Definition of Done](../00-governance/definition-of-done.md) — requires action tracker update

## Used By / Affects

Definition of Done verification, project audit trail.

## Risks If Changed

LOW — this is an append-only log.
